NoteShrink
==========

This script makes scans and handwritten notes more beautiful! The script clears synopses and scans from "dirt", improves their readability and reduces the size of the image.

## Requirements

 - Python 3
 - NumPy 1.10 or later
 - SciPy
 - ImageMagick
 - Image module Pillow

## Usage

```
./noteshrink.py IMAGE1 [IMAGE2 ...]
```
